## English version

**Post 1 (attach the video)**

> I built an AI chief-of-staff for my Mac. It records my meetings, turns
> requests into proposal cards, and runs background Claude agents to do the
> work. I only do two things: approve and accept.

**Post 2**

> How it works:
> 1. screenpipe records screen + meetings locally
> 2. radars scan Obsidian, Slack and Gmail for things people ask me to do
> 3. each becomes a card: plan, acceptance criteria, cost
> 4. one click, and a Claude agent works in an isolated git worktree, ships a draft PR

**Post 3**

> Everything is local-first. Recordings, transcripts and the requirement
> registry never leave my Mac. The app only reads one JSON file and writes my
> approvals to another.

**Post 4**

> Code: github.com/Wan-ZL/zelin-ai-assistant
> macOS 14+, with iOS / Linux / Windows betas. Source available (FSL-1.1-MIT).
> All data in the video is fictional demo seed data.
> Music: "Voxel Revolution" Kevin MacLeod (incompetech.com), CC BY 4.0
